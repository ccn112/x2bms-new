## Bắt đầu

### Đăng nhập
Truy cập hệ thống, đăng nhập bằng email + mật khẩu được cấp. Tuỳ vai trò, bạn vào một trong ba không gian:
- **Ban quản lý (BQL)** — quản lý 1 dự án: cư dân, căn hộ, xe/thẻ, phí, phản ánh…
- **Công ty vận hành (Tenant/HQ)** — tổng hợp nhiều dự án.
- **Quản trị nền tảng (SuperAdmin)** — cấu hình toàn hệ thống.

### Thanh điều hướng
- **Sidebar trái**: nhóm chức năng theo module.
- **Breadcrumb** (đường dẫn) trên đầu trang: mục **màu xanh** là bấm được để quay lại; mục cuối (màu xám) là trang hiện tại.
- **Thanh công cụ mỗi màn**: tối đa 3 nút chính hay dùng; các thao tác khác nằm trong nút **"Thao tác khác" (⋯)**. Màu nút gợi ý ý nghĩa: **xanh dương/vàng** = tạo/chính, **xanh lá** = duyệt/mở, **đỏ** = khoá/xoá, **vàng cam** = bảo mật (đặt lại mật khẩu).

Chi tiết từng nghiệp vụ: xem các trang con trong mục lục.
