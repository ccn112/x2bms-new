## Data source (no hardcoded values in view)

| Screen figure | Query |
|---|---|
| Tỷ lệ thu phí (96.2%) | `billing_periods` current: `collected_amount / billed_amount` |
| Đã thu trong tháng (2,45 tỷ) | current `billing_periods.collected_amount` |
| Phản ánh chờ xử lý (56) | `feedback_requests` where status ∈ pending |
| Cảnh báo SLA (18) | `sla_events` where status = open |
| Công nợ đến hạn (96 tr) | `debts` where is_overdue sum(amount) |
| Bar chart thu phí | `billing_periods` collected by month |
| Donut phản ánh (132) | `feedback_categories` withCount(`feedback_requests`) |
| Bảng việc cần xử lý | `work_orders` open, with department |
| Hiệu suất bộ phận | `work_orders` done / total per department |
| Cảnh báo list | `ioc_alerts` open |
| AI panel | `ai_suggestions` context=operational_dashboard |
| Audit footer | latest `audit_logs` |
