# WEB-02-02 — Hồ sơ căn hộ

- **Source image:** `handoff/.../WEB-02_Cư_dân_and_Căn_hộ/images/WEB-02-02__Hồ_sơ_căn_hộ.png`
- **Type:** record/detail view (+ list)
- **Render strategy:** Custom Pages — `App\Livewire\ApartmentDirectory` (`GET /apartments`, list) and `App\Livewire\ApartmentProfile` (`GET /apartments/{apartment}/profile`, detail via route-model binding). Livewire + Blade + X2 components.
- **X2 components:** X2AdminShell, X2ActionBar, X2KpiCard, X2SectionCard, X2DataTable, X2StatusBadge, X2AiPanel; inline SVG donut (collection rate).
- **Canonical entities/tables:** `apartments`, `floors`, `buildings`, `resident_apartment_relations`, `residents`, `vehicles`, `access_cards`, `statements`, `debts`.
- **Data source (no hardcode):** apartment + floor + building; members from relations; vehicles/cards by apartment; collection rate = `statements.paid/total`; overdue from `debts`; AI from generated context.
- **Sample data shown:** Mã căn, Diện tích, Chủ sở hữu, Người trong hộ, Phương tiện, Thẻ ra vào, Tỷ lệ thu phí, Công nợ.
- **Seed:** 120 apartments (floor + area), 108 vehicles, 120 cards, statements for first 12 apartments, overdue debts.
- **RBAC/tenancy:** auth + `BelongsToTenant`; “Chỉnh sửa” links to Filament `ApartmentResource` edit.
- **Acceptance:** `/apartments` lists căn; `/apartments/{id}/profile` shows sections + donut + vehicles/cards.
- **Screenshot test:** `tests/Browser/web-02.spec.ts` → `WEB-02-02 apartment directory + profile`.
