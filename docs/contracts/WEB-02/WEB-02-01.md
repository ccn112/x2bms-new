# WEB-02-01 — Danh sách cư dân

- **Source image:** `handoff/.../WEB-02_Cư_dân_and_Căn_hộ/images/WEB-02-01__Danh_sách_cư_dân.png`
- **Type:** list view
- **Render strategy:** Custom Page — Livewire `App\Livewire\ResidentDirectory` + Blade + X2 components (live search). Route `GET /residents` (name `residents`, auth).
- **X2 components:** X2AdminShell (PageShell/Sidebar/Topbar/AuditFooter), X2ActionBar, X2KpiCard ×4, X2SectionCard, X2DataTable, X2StatusBadge.
- **Canonical entities/tables:** `residents`, `resident_apartment_relations`, `apartments`, `floors`, `buildings`.
- **Data source (no hardcode):** KPIs from `Resident::count()`, relation role counts, pending `resident_approval_requests`. Rows = primary `resident_apartment_relations` with resident + apartment + floor. Live `search` filters name/phone.
- **Sample data shown:** Họ tên + mã, SĐT, Email, Vai trò (Chủ sở hữu/Người thuê/Thành viên), Trạng thái xác thực, Tòa/Tầng/Căn.
- **Seed:** 120 residents (roles ~70/22/8%), 8 pending approvals → KPIs 120 / 88 / 24 / 8.
- **RBAC/tenancy:** auth + `BelongsToTenant` scope; future permission `resident.view`.
- **Acceptance:** after `migrate:fresh --seed`, `/residents` shows KPIs + 15/120 residents with badges + search.
- **Screenshot test:** `tests/Browser/web-02.spec.ts` → `WEB-02-01 resident directory`.
